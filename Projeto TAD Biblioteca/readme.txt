Este projeto foi feito em linguagem C como um trabalho para a disciplina Algoritmos e Estruturas de Dados I em 2021. 
Nele, é simulado uma biblioteca virtual onde é possível cadastrar livros, pesquisar livros cadastrados, alterar 
dados já cadastrados e fazer a impressão dos dados. Também é possível fazer a mesma coisa com usuários e 
com empréstimos de livros. Todos os arquivos na pasta contidas são necessários para o funcionamento do programa.
